# N無人士入名校攻略研究筆記

## 來源 1: Champimom - Team A 和 Team B 分別
URL: https://www.champimom.com/posts/考小學_TeamA_TeamB

### 什麼是 Team A 和 Team B？

在申請直資和私立小一時，學校將申請者分為兩類：

**Team A（有關係的申請者）包括：**
- 現在有兄姊在該小學或聯繫中學就讀
- 兄姊畢業於該小學或聯繫中學
- 父母畢業於該小學或聯繫中學
- 父母在該小學或聯繫中學是現任教職員或校內員工
- 有直系親屬在該學校為校董會或舊生會成員

**Team B（N無人士）：**
- 沒有上述任何關係的申請者
- 俗稱「N無」：無宗教、無校網、無關係、無證書

### Team A 和 Team B 面試有何不同？

大部份情況下，面試內容及流程完全相同，只是甄選過程可能有些不一樣。

**設有兩輪面試的學校：**
- 聖保羅男女中學附屬小學
- 拔萃男書院附屬小學
- 拔萃女小學
- 英華小學
- 陳守仁小學
- 香港浸會大學附屬學校王錦輝中小學

這些學校在報名表上會詢問家長及小朋友是否有以上背景。有關係的學生非常大機會甚至保證有第二輪面試機會。

### 取錄比例

唯一公開數據的學校是**聖保羅男女中學附屬小學**：
- 有關係的學生：上限 70%
- 沒有關係的學生：最少 30%

### N無人士（Team B）入名校策略

根據日朗坊面試培訓課程總監 Raymond Sir 分析：

> 有關係和沒有關係對於取錄結果並沒有直接關係。過往很多「N無」（Team B）學生家長從小培養小朋友的良好學習動機和言談舉止，加上父母能夠給校方一個正向教育家庭的良好印象，這些因素比起有沒有關係更為重要。

**N無人士成功關鍵：**
1. 從小培養小朋友的良好學習動機
2. 培養良好的言談舉止
3. 父母展現正向教育家庭的良好印象
4. 做好充分的面試準備

## 小一入學計分制策略（適合 N 無人士）

### 自行分配學位計分制

| 分數 | 條件 |
|------|------|
| 20分 | 有兄/姊在該小學就讀，或父/母在該小學任職 |
| 10分 | 父/母為該小學的校董 |
| 10分 | 兄/姊為該小學的畢業生 |
| 10分 | 父/母為該小學的畢業生 |
| 10分 | 首名出生子女 |
| 5分 | 與該校有相同宗教信仰 |
| 5分 | 父/母為該小學主辦社團的成員 |

### N無人士策略建議：

**策略一：選擇無宗教小學**
- 15分或以下的學童一般無法取得宗教分
- 選擇無宗教背景的小學可減低競爭
- 首選官立小學（由政府營運，無宗教及辦學團體背景）

**策略二：報讀直資/私立小學**
- 不受校網限制
- 不受計分制影響
- 純粹以面試表現決定

**策略三：叩門準備**
- 統一派位結果公布後
- 以「後補生」方式爭取心儀學校取錄
- 需要準備 Portfolio 和面試
