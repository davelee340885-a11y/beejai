# 香港法例研究：使用學校商標和網站資訊的法律問題

## 研究日期
2024年12月27日

## 研究來源
1. 香港知識產權署 - 商標條例常見問題
   URL: https://www.ipd.gov.hk/tc/trade-marks/faqs/trade-marks-ordinance/index.html

## 關鍵發現

### 一、商標侵權的定義

根據香港知識產權署的說明，關於「我應採取什麼策略來保護我在香港特區及中國內地的商標？」的問題：

**未經許可使用他人商標的情況：**

1. **採用一個與別不同和不會侵犯他人權益的商標**
   - 侵犯馳名商標的意思可包括使用該商標或類似的商標於不相同的貨品
   - 即使你的產品與該馳名商標的產品屬於不同，你亦可能會被指侵犯他人權利

2. **內地的商標註冊及保障制度與香港特區並不相同**
   - 因此你應分別在兩地註冊你的商標
   - 如只在內地註冊某商標，該商標並不會自動在香港特區獲得保護

3. **同時把商標的英文名稱(翻譯或音譯)註冊以便在本地使用**
   - 可以增加商標的吸引力和擴大商標的保護範圍

4. **應盡早為你的商標註冊**
   - 以確保你可在業務上使用該商標
   - 並可對任何侵權行為即時採取行動

5. **如你準備為使用商標此**
   - 你應訂立一個完善的商標計劃

6. **經常查閱網上的香港知識產權公報**
   - 以查看是否有其他人申請註冊可能導致公眾對你的註冊商標引起混淆的商標
   - 如有的話，可提出反對

7. **如有人在香港特區侵犯你的商標**
   - 你可考慮根據香港特區《商標條例》採取行動
   - 進行民事訴訟

8. **如發現你在香港特區的貨品或服務的商譽及聲譽因受到別人假冒而蒙受損失**
   - 你應採取法律行動

9. **如商標侵權涉及刑事侵權行為並在香港特區發生**
   - 你可考慮向香港海關及商標調查科舉報
   - 你須向海關提供詳細資料，證明你擁有有關商標
   - 並提出採取緝私的請求
   - 詳情請參閱以下網址：https://www.customs.gov.hk/tc/service-enforcement-information/intellectual-property-rights-protection/index.html

10. **如侵權行為在內地發生**
    - 你應按照內地法律要求行政處理或採取法律行動
    - 如某間香港公司以你的註冊商標名，以掩飾其在內地的侵權行為
    - 你可以根據內地法律採取行動，制止有關侵權行為

11. **最重要的一點**
    - 是你應就版權、商標或註冊外觀設計等知識產權的各種權益
    - 徵詢知識產權律師或代理人的專業意見
    - 為你的產品或服務提供保障

12. **就其他有關的商業保護問題尋求專業人士的意見**
    - 包括以你的註冊商標、商業和公司名稱註冊為域名
    - 以保護其在互聯網上的使用
    - 香港域名註冊有限公司(https://www.hkirc.hk)為一非牟利公司
    - 負責有關 HK 結尾的域名註冊事宜

## 二、關鍵法律問題

### 1. 使用學校商標的風險

根據上述資料，**未經許可使用他人的商標作為商號或公司名稱屬侵犯商標行為**。這意味著：

- 如果學校的名稱或標誌已註冊為商標，在網站上展示這些商標可能構成侵權
- 即使是用於資訊展示目的，仍需考慮是否符合「合理使用」原則

### 2. 需要進一步研究的問題

1. **合理使用（Fair Use）原則**
   - 香港商標法是否有類似的合理使用條款？
   - 用於教育資訊平台的展示是否屬於合理使用？

2. **版權法對網站內容的保護**
   - 從學校網站獲取的資訊（如學費、課程資料）是否受版權保護？
   - 轉載這些資訊是否需要授權？

3. **資料保護條例**
   - 展示學校資訊是否涉及個人資料保護問題？

## 三、商標的誠實使用（Fair Use）

### 來源：社區法網（CLIC）
URL: https://www.clic.org.hk/zh/topics/intellectualProperty/trade_marks/infringement/trade_mark_in_advertising_materials

### 《商標條例》第21條規定

**關鍵內容：**

根據《商標條例》第21條，**於廣告中按照在工業或商業事宜中的誠實做法使用已註冊的商標並不構成侵權作為**。

在決定有關使用是否按照在工業或商業事宜中的誠實做法而作出時，法院可以考慮其認為有關的因素，尤其包括：

1. **該使用是否對該商標構成不公平的利用**
2. **該使用是否對該商標的顯著特性或聲譽造成損害**
3. **該使用是否會欺騙公眾**

### 對 BeeJAI 平台的意義

這條法例為教育資訊平台使用學校商標提供了一定的法律依據，但需要滿足以下條件：

1. **誠實使用**：必須是按照誠實的商業做法使用
2. **不構成不公平利用**：不能利用學校商標來獲取不當利益
3. **不損害聲譽**：不能損害學校商標的顯著特性或聲譽
4. **不欺騙公眾**：不能讓公眾誤以為平台與學校有官方關係

### 風險評估

**中等風險**：雖然第21條提供了誠實使用的豁免，但仍需要：
- 清楚標示資訊來源
- 避免讓用戶誤以為平台與學校有官方合作關係
- 不能用學校商標作為平台的主要賣點
- 必須在用戶條款中明確免責聲明


## 四、版權的例外情況（合理使用）

### 來源：香港知識產權署
URL: https://www.ipd.gov.hk/tc/copyright/what-is-copyright/index.html

### 版權例外情況的規定

**關鍵內容：**

為了在版權擁有人的權益與公眾利益之間取得平衡，**法律容許若干例外情況**。只有當某項作品的實質部分被人採用時，才算是侵犯版權。這是個平衡而非絕對的問題。舉例來說，假如某某擷取網內某節受錄音上的某句歌詞人抄襲，即使有關的樂句很短，也可構成侵權行為。

**在若干條件的規限下，法例容許公眾人士合理使用版權作品，以便進行：**

1. **研究**
2. **私人研習**
3. **批評**
4. **評論**
5. **新聞報導**

**也准許圖書館及學校合理使用版權作品。**

**重要提醒：**
不過，使用者仍須小心，以免侵犯版權。舉例來說，複如某本書的大部分內容，可能會構成侵權行為。

### 對 BeeJAI 平台的意義

這條規定為教育資訊平台提供了一定的法律依據，但需要注意：

1. **新聞報導目的**：如果平台是以新聞報導的方式提供學校資訊，可能符合合理使用
2. **研究和評論目的**：如果是用於教育研究或評論，也可能符合合理使用
3. **不能複製大部分內容**：不能直接複製學校網站的大部分內容
4. **必須註明出處**：使用時必須註明資料來源

### 風險評估

**中低風險**：如果平台：
- 僅提供學校的基本資訊（如學費、課程、地址等事實性資料）
- 註明資料來源
- 不複製學校網站的大部分內容
- 以新聞報導或教育研究的方式呈現

則侵犯版權的風險相對較低。

## 五、綜合法律分析

### 1. 使用學校商標的法律風險

**風險等級：中等**

**法律依據：**
- 《商標條例》第21條允許「按照誠實做法」使用已註冊商標
- 但需確保不構成不公平利用、不損害聲譽、不欺騙公眾

**建議措施：**
1. 在網站顯眼位置加入免責聲明，說明平台與學校無官方關係
2. 使用學校商標時，僅作為識別用途，不作為平台的主要賣點
3. 避免使用可能讓公眾誤以為有官方合作關係的表述
4. 考慮使用學校名稱的文字描述，而非直接使用學校的圖形商標

### 2. 轉載學校網站資訊的法律風險

**風險等級：中低**

**法律依據：**
- 《版權條例》允許合理使用版權作品進行新聞報導、研究、評論
- 事實性資料（如學費、地址、課程名稱）通常不受版權保護
- 但學校網站的整體設計、照片、文章等仍受版權保護

**建議措施：**
1. 僅轉載事實性資料，避免直接複製學校網站的文章或描述
2. 用自己的文字重新整理和呈現資訊
3. 註明資料來源和更新日期
4. 不複製學校網站的照片、設計元素
5. 在用戶條款中加入免責聲明，說明資料僅供參考

### 3. 最佳實踐建議

**為了最大程度降低法律風險，BeeJAI 平台應該：**

1. **使用公開數據源**
   - 優先使用香港教育局公開的學校資料（如教育局網站的學校名單）
   - 這些政府公開數據通常可以免費使用

2. **建立資料收集流程**
   - 聘請專人致電或電郵學校，直接向學校索取最新資訊
   - 取得學校的口頭或書面同意展示其資訊

3. **完善免責聲明**
   - 在用戶條款中明確說明所有資料僅供參考
   - 說明平台與學校無官方關係
   - 建議用戶直接聯絡學校確認資訊

4. **建立投訴機制**
   - 如學校要求移除其資訊或商標，應立即配合
   - 設立專門的聯絡渠道處理學校的投訴

5. **考慮取得授權**
   - 對於重點學校，可考慮主動聯絡並取得書面授權
   - 建立合作關係，讓學校主動提供資訊

### 4. 結論

**總體風險評估：中等，但可控**

只要 BeeJAI 平台：
- 遵循誠實使用原則
- 完善免責聲明
- 註明資料來源
- 不誤導公眾
- 建立投訴處理機制

則法律風險是可控的。建議在正式上線前諮詢知識產權律師，以確保所有法律條款和免責聲明都符合香港法例要求。

2. **教育局**

   - 了解學校對於資訊平台的態度和期望
