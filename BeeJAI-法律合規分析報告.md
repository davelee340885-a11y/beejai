# BeeJAI 升學資訊平台法律合規分析報告

**報告日期：** 2024年12月27日  
**撰寫：** Manus AI  
**目的：** 分析 BeeJAI 平台使用學校商標和轉載學校網站資訊的法律風險

---

## 執行摘要

BeeJAI 作為香港升學資訊平台，在展示學校商標和轉載學校網站資訊時，需要遵守香港《商標條例》（第559章）和《版權條例》（第528章）的相關規定。經過深入研究香港知識產權法例，本報告得出以下結論：

**總體風險評估：中等，但可控**

只要平台遵循誠實使用原則、完善免責聲明、註明資料來源、不誤導公眾並建立投訴處理機制，法律風險是可以有效控制的。本報告提供了詳細的合規建議和最佳實踐指引。

---

## 一、法律框架分析

### 1.1 商標法：《商標條例》（第559章）

根據香港知識產權署的說明[1]，未經許可使用他人的商標可能構成侵權行為。然而，《商標條例》第21條提供了「誠實使用」的豁免條款[2]。

**第21條關鍵規定：**

於廣告中按照在工業或商業事宜中的誠實做法使用已註冊的商標並不構成侵權作為。在決定有關使用是否按照誠實做法而作出時，法院可以考慮以下因素：

1. **該使用是否對該商標構成不公平的利用**
2. **該使用是否對該商標的顯著特性或聲譽造成損害**
3. **該使用是否會欺騙公眾**

**對 BeeJAI 的意義：**

這條法例為教育資訊平台使用學校商標提供了一定的法律依據，但必須確保使用方式符合「誠實做法」的標準，不能讓公眾誤以為平台與學校有官方合作關係。

### 1.2 版權法：《版權條例》（第528章）

根據香港知識產權署的說明[3]，版權法在保護版權擁有人權益的同時，也容許若干例外情況。

**合理使用（Fair Dealing）條款：**

在若干條件的規限下，法例容許公眾人士合理使用版權作品，以便進行：

- **研究**
- **私人研習**
- **批評**
- **評論**
- **新聞報導**

也准許圖書館及學校合理使用版權作品。

**重要限制：**

使用者仍須小心，以免侵犯版權。舉例來說，複製某本書的大部分內容，可能會構成侵權行為。

**對 BeeJAI 的意義：**

如果平台是以新聞報導或教育研究的方式提供學校資訊，可能符合合理使用條款。但必須注意不能直接複製學校網站的大部分內容，並且必須註明資料來源。

## 二、風險評估

### 2.1 使用學校商標的法律風險

**風險等級：中等**

| 風險因素 | 風險描述 | 風險等級 |
|---------|---------|---------|
| 商標侵權 | 未經許可使用學校已註冊商標 | 中 |
| 誤導公眾 | 讓用戶誤以為平台與學校有官方關係 | 高 |
| 不公平利用 | 利用學校商標獲取不當商業利益 | 中 |
| 損害聲譽 | 不當使用導致學校聲譽受損 | 低 |

**法律依據：**

根據《商標條例》第21條，「按照誠實做法」使用已註冊商標不構成侵權。但需確保不構成不公平利用、不損害聲譽、不欺騙公眾。

**風險緩解措施：**

1. 在網站顯眼位置加入免責聲明，說明平台與學校無官方關係
2. 使用學校商標時，僅作為識別用途，不作為平台的主要賣點
3. 避免使用可能讓公眾誤以為有官方合作關係的表述
4. 考慮使用學校名稱的文字描述，而非直接使用學校的圖形商標

### 2.2 轉載學校網站資訊的法律風險

**風險等級：中低**

| 資訊類型 | 版權保護程度 | 風險等級 | 建議做法 |
|---------|-------------|---------|---------|
| 事實性資料（學費、地址） | 不受版權保護 | 低 | 可直接使用 |
| 課程名稱、學制 | 不受版權保護 | 低 | 可直接使用 |
| 學校描述文章 | 受版權保護 | 高 | 避免直接複製 |
| 學校照片、圖片 | 受版權保護 | 高 | 避免使用 |
| 學校網站設計 | 受版權保護 | 高 | 避免模仿 |

**法律依據：**

根據《版權條例》，合理使用版權作品進行新聞報導、研究、評論是允許的。事實性資料（如學費、地址、課程名稱）通常不受版權保護，但學校網站的整體設計、照片、文章等仍受版權保護。

**風險緩解措施：**

1. 僅轉載事實性資料，避免直接複製學校網站的文章或描述
2. 用自己的文字重新整理和呈現資訊
3. 註明資料來源和更新日期
4. 不複製學校網站的照片、設計元素
5. 在用戶條款中加入免責聲明，說明資料僅供參考

## 三、合規建議

### 3.1 使用公開數據源

**優先級：高**

**建議措施：**

優先使用香港教育局公開的學校資料，這些政府公開數據通常可以免費使用，法律風險最低。

**可用的公開數據源：**

1. **教育局學校名單**
   - 包含學校名稱、地址、電話、學制等基本資訊
   - 網址：https://www.edb.gov.hk

2. **教育局學校概覽**
   - 包含學校簡介、辦學宗旨、課程特色等
   - 每年更新一次

3. **教育局統計數據**
   - 包含學校學生人數、教師人數等統計資料

**實施步驟：**

1. 下載教育局公開的學校資料 CSV 檔案
2. 建立自動化腳本定期更新資料
3. 在網站註明資料來源為「香港教育局」

### 3.2 建立資料收集流程

**優先級：中**

**建議措施：**

聘請專人致電或電郵學校，直接向學校索取最新資訊，並取得學校的口頭或書面同意展示其資訊。

**實施步驟：**

1. **建立學校聯絡資料庫**
   - 記錄每間學校的聯絡人、電話、電郵

2. **設計標準化的資料收集表格**
   - 包含學費、課程、入學要求等標準欄位

3. **定期更新資料**
   - 每學年開始前聯絡學校更新資訊
   - 記錄資料更新日期

4. **取得使用同意**
   - 在聯絡學校時，說明平台用途
   - 取得學校口頭或書面同意展示其資訊
   - 記錄同意日期和方式

### 3.3 完善免責聲明

**優先級：高**

**建議措施：**

在用戶條款、私隱政策和網站顯眼位置加入完整的免責聲明。

**免責聲明應包含的內容：**

1. **資料準確性免責**
   ```
   本平台所提供的學校資訊僅供參考，我們已盡力確保資訊的準確性，
   但不保證資訊完全準確、完整或最新。用戶應直接聯絡學校確認所有資訊。
   ```

2. **無官方關係聲明**
   ```
   BeeJAI 是一個獨立的升學資訊平台，與任何學校、教育機構或政府部門
   均無官方合作關係。本平台展示的學校商標和資訊僅作識別和資訊提供用途。
   ```

3. **用戶責任聲明**
   ```
   用戶根據本平台提供的資訊作出任何決定或採取任何行動，均需自行承擔責任。
   本平台對於因使用或依賴本平台資訊而導致的任何損失或損害，概不負責。
   ```

4. **知識產權聲明**
   ```
   本平台展示的學校名稱、商標和標誌均為相關學校的知識產權。
   本平台僅按照誠實做法使用這些商標作為識別用途，不代表與學校有任何合作關係。
   如學校要求移除其資訊或商標，請聯絡我們，我們將立即配合。
   ```

**實施位置：**

1. 用戶條款頁面（詳細版本）
2. 私隱政策頁面（相關部分）
3. 網站頁腳（簡短版本）
4. 學校詳情頁面（簡短提示）
5. 登入前條款接受對話框（已實施）

### 3.4 建立投訴處理機制

**優先級：高**

**建議措施：**

設立專門的聯絡渠道處理學校的投訴，如學校要求移除其資訊或商標，應立即配合。

**實施步驟：**

1. **設立專用電郵地址**
   - 例如：school-feedback@beejai.com
   - 在網站顯眼位置展示

2. **建立投訴處理流程**
   - 收到投訴後24小時內回覆
   - 48小時內完成調查和處理
   - 記錄所有投訴和處理結果

3. **移除資訊的標準程序**
   - 確認投訴來源（確保是學校官方代表）
   - 立即移除相關資訊或商標
   - 向投訴方確認已完成移除
   - 記錄移除原因和日期

4. **定期檢討**
   - 每季度檢討投訴情況
   - 分析常見投訴原因
   - 改進資料收集和展示方式

### 3.5 考慮取得授權

**優先級：低至中**

**建議措施：**

對於重點學校（如排名前50的熱門學校），可考慮主動聯絡並取得書面授權，建立合作關係。

**實施步驟：**

1. **篩選重點學校**
   - 根據用戶搜尋量和關注度
   - 優先聯絡熱門學校

2. **設計合作方案**
   - 提供免費的學校資訊展示服務
   - 協助學校更新資訊
   - 提供數據分析報告（如頁面瀏覽量）

3. **取得書面授權**
   - 準備標準授權協議範本
   - 明確授權範圍和期限
   - 記錄授權狀態

4. **建立合作關係**
   - 定期向學校提供數據報告
   - 優先展示已授權學校
   - 建立長期合作關係

## 四、最佳實踐指引

### 4.1 學校商標使用指引

**允許的使用方式：**

1. **文字形式的學校名稱**
   - 優先使用學校的中英文全名
   - 例如：「拔萃女書院 Diocesan Girls' School」

2. **小尺寸的學校徽章**
   - 僅用於學校列表和搜尋結果
   - 尺寸不超過 50x50 像素
   - 作為識別用途，不作為裝飾

3. **註明資料來源**
   - 在學校詳情頁面底部註明
   - 例如：「資料來源：學校官方網站 / 香港教育局」

**禁止的使用方式：**

1. **大尺寸或突出展示學校商標**
   - 不在首頁橫幅使用學校商標
   - 不在廣告中使用學校商標

2. **暗示官方合作關係**
   - 不使用「合作學校」「官方推薦」等字眼
   - 不在學校商標旁加上「認證」「授權」標誌

3. **修改學校商標**
   - 不改變學校商標的顏色、比例或設計
   - 不在學校商標上添加其他元素

### 4.2 學校資訊轉載指引

**可以轉載的資訊：**

| 資訊類型 | 示例 | 注意事項 |
|---------|------|---------|
| 基本資料 | 學校名稱、地址、電話、電郵 | 註明來源 |
| 學制資料 | 小學、中學、幼稚園、國際學校 | 註明來源 |
| 學費資料 | 年度學費、雜費 | 註明學年和來源 |
| 課程資料 | IB、DSE、IGCSE | 註明來源 |
| 入學要求 | 年齡要求、面試流程 | 用自己的文字整理 |
| 統計數據 | 學生人數、教師人數 | 註明來源和年份 |

**不應轉載的資訊：**

| 資訊類型 | 原因 | 替代方案 |
|---------|------|---------|
| 學校簡介文章 | 受版權保護 | 用自己的文字重寫 |
| 學校照片 | 受版權保護 | 使用公開圖庫或自行拍攝 |
| 校長的話 | 受版權保護 | 不展示或取得授權 |
| 學校宣傳影片 | 受版權保護 | 連結到學校官網 |
| 學校活動報導 | 受版權保護 | 用自己的文字整理 |

### 4.3 資料來源標示指引

**標準格式：**

```
資料來源：香港教育局
最後更新：2024年9月
```

或

```
資料來源：學校官方網站
最後更新：2024年9月
本平台與學校無官方關係，資訊僅供參考
```

**標示位置：**

1. 學校詳情頁面底部
2. 學費資料下方
3. 課程資料下方

## 六、實施時間表

### 第一階段：立即實施（1-2週）

**優先級：高**

1. **更新用戶條款和私隱政策**
   - 加入知識產權聲明
   - 加強免責聲明
   - 完成法律審核

2. **在網站顯眼位置加入免責聲明**
   - 首頁頁腳
   - 學校詳情頁面
   - 關於我們頁面

3. **設立投訴處理機制**
   - 設立專用電郵地址
   - 建立投訴處理流程
   - 培訓客服人員

3. **註明資料來源**
   - 在所有學校資訊下方加入來源標示
   - 註明最後更新日期

**優先級：低至中**

1. **聯絡重點學校取得授權**

   - 設計合作方案
   - 主動聯絡學校

## 九、結論

BeeJAI 升學資訊平台在展示學校商標和轉載學校網站資訊時，面臨一定的法律風險，但這些風險是可以通過適當的合規措施有效控制的。

**關鍵成功因素：**

1. **嚴格遵守「誠實使用」原則**
   - 僅將學校商標用於識別目的
   - 不暗示與學校有官方合作關係
   - 不利用學校商標獲取不當利益

2. **完善免責聲明和用戶條款**
   - 清楚說明資料僅供參考
   - 明確平台與學校無官方關係
   - 要求用戶自行承擔使用資訊的責任

3. **建立透明的資料收集和更新機制**
   - 優先使用政府公開數據
   - 直接向學校收集資訊並取得同意
   - 定期更新資料並註明來源

4. **建立快速回應的投訴處理機制**
   - 48小時內處理學校投訴
   - 主動配合學校移除資訊的要求
   - 建立長期合作關係

5. **定期進行法律審核**
   - 每季度檢討合規情況
   - 每年諮詢知識產權律師
   - 及時更新法律條款

**最終建議：**

在正式上線前，強烈建議 BeeJAI 平台：

1. **聘請知識產權律師**審核所有法律條款和免責聲明
2. **與香港教育局溝通**確認使用公開數據的具體要求
3. **與學校協會或校長會溝通**了解學校對資訊平台的期望
4. **建立完善的合規流程**確保所有員工了解法律要求

只要遵循本報告的建議，BeeJAI 平台可以在合法合規的前提下，為香港家長提供優質的升學資訊服務。
